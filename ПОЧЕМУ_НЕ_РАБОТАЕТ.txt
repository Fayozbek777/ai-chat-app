Проект использует OpenAI API для взаимодействия с моделью GPT. 
Во время тестирования я сам исчерпал весь доступный лимит токенов. 
Из-за этого API перестал возвращать ответы и выдаёт ошибки авторизации или превышения лимита. 
После восстановления лимита или начала нового биллингового цикла API снова будет работать.  


Скоро будет добавлено решение/фикс, чтобы это ограничение учитывалось автоматически.
